# 🚗 Car Price Prediction System

## 📌 Overview

This project predicts the price of a used car based on:

* Model Year
* Kilometers Driven
* Fuel Type

It also includes a comparison feature for two cars.

---

## 🚀 Features

* Single car price prediction
* Compare two cars
* Explanation of prediction
* Premium red-black UI

---

## 🧠 Tech Stack

* Python
* Flask
* Scikit-learn
* HTML, CSS

## ▶️ How to Run

1. Navigate:

```
cd OIBSIP_ds_2
```

2. Install dependencies:

```
pip install -r requirements.txt
```

3. Train model:

```
python train_model.py
```

4. Run app:

```
python app.py
```

5. Open:

```
http://127.0.0.1:5000
```
